Esta aplicacion basicamente lo que hace es un CRUD con una base de datos donde estan almacenados los productos del almacen, por lo tanto podrá leer los productos en la BBDD, insertar nuevos productos, modificar datos en los productos ya cargados,como tambien eliminarlos. Todo es en base al campo clave de la tabla productos, o sea CÓDIGOARTÍCULO.

LENGUAJES


LENGUAJES Y TECNOLOGÍAS UTILIZADOS


PARA EL BACK-END:

  • Patron de arquitectura MVC.
  • Modelo y controlador: estan completamente programados con JEE.
  • El controlador es un Servlet java que pasa al modelo objetos Producto(POJO)
  • El modelo se maneja con consultas preparadas(PreparedStatement) y JDBC, accede a la BBDD por un pool de conexiones.
  • El servlet recibe un parametro desde la vista que evalúa por un condicional IF y realiza la acción correspondiente según el caso.

PARA EL FRONT-END

  • La vista principal esta creada con HTML5 y CSS.
  • Algunas paginas estan hechas con JSP
  • Los formularios de actualizar productos usan librerias jstl(Tags)
  • Los formularios envian los datos al controlador(servlet) por el metodo GET
